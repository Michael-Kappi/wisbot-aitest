AI Documentation Assistant Demo

This is a proof-of-concept demonstrating how an AI chatbot can help users find information from product documentation without manually searching through files.

How it works

  • Documentation files are loaded from the docs/ folder on the server
  • The AI assistant uses these documents to answer your questions
  • Click the chat icon in the top-right corner to start a conversation

Try asking

  • "How do I select different tyres for the truck and trailer?"
  • "What weight limits does the validation engine check?"
  • "How do I compare two configurations?"
